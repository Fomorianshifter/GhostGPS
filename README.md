# GhostGPS

A mobile app for simulating GPS driving behavior to enhance privacy and provide useful features like cops ahead reporting, with integration into Google Maps and Waze.

## Description

This app simulates driving to a location while adhering to posted speed limits. It includes cops ahead reporting and works alongside Google Maps and Waze.

Note: This is completely legal, as similar features are offered by Google, Waze, and other GPS companies.

## Features
- Simulates driving route with speed limit compliance.
- Cops ahead alerts.
- Integration with Google Maps and Waze for side-by-side functionality.
